# Content Flywheel: ACB 内容驱动 Lead 飞轮

> Houzz 模式跑通的命门：**ACB（Aqara Certified Builder）是否愿意持续发布内容**。
> Builder 平台必须显式设计这个机制——做归因透明、做激励闭环。
> 内容来源天然包含 **Studio 部署数据**：每个真实 Studio 项目都是潜在 Showcase。

---

## 飞轮路径

```
ACB 完成项目（Studio 部署 + Personas 交付）
   ↓
AI 起草 Showcase（从 Studio 空间图谱 / 设备 / 插件自动提取）
   ↓
ACB 审阅发布
   ↓
AI 打标推荐到 Discover
   ↓
用户浏览 → 加入 Ideabook
   ↓
AI 触达"落地到我家"
   ↓
Lead 派给匹配 ACB（按 Badge + 区域 + Studio 部署经验）
   ↓
AI 协助 ACB 回复报价
   ↓
成交 → 部署 Studio → 交付 Personas
   ↓
新内容（回到顶部）
```

---

## 激励 ACB 发布内容的四类机制（比 Houzz 更狠）

### 1. 内容 → Lead 归因可见

> ACB 看到："这个 Showcase 带来 8 咨询 / 3 成交 / 2 个新 Studio 部署"。

每条 Showcase / 每张图 / 每个 Persona 案例都有归因看板。

### 2. 内容 → 插件 + 订阅销售归因

Showcase 里嵌的插件被买、引出的客户订阅 Aqara AI（Studio 月费），ACB 持续拿分成（10–15%）。

> 这是 Houzz 没有的维度，**让内容变成长期资产**。

### 3. 内容 → 排名加成

发布频次 + 质量影响 ACB Directory 排序（含 Builder 公开 Profile + Builder Network 派单权重）。

> 算法**透明**：ACB 知道发什么、怎么发能涨权重。

### 4. 内容 → Aqara 收购

明星案例基金，**买断 / 分润**高质量内容做官方推广。

---

## 降低发布门槛的核心：AI 自动起草

> 目标：ACB 发布 Showcase 从 **4 小时 → 15 分钟**。

| 步骤 | AI 做什么 |
|---|---|
| 数据提取 | 从已完成项目自动提取 **Studio 空间图谱**可视化 |
| 清单生成 | 自动列出设备 / 插件 / Personas |
| 故事生成 | 自动生成"设计亮点"和"为什么这样设计" |
| 自动打标 | 风格 / 户型 / 区域 / Persona / Badge 标签 |
| 隐私脱敏 | 人脸 / 地址 / 敏感物品自动模糊 |
| **ACB 审阅** | **必须**最终人工审阅，不能直接发布 |

---

## 飞轮启动期：种子内容

> 避免冷启动失败。

- Aqara 内部 + 邀请区域伙伴 + 首批 ACB 共创 **50–100 个 Showcase** 启动
- 每个种子 Showcase 必须对应一个真实的 Studio 部署（保证内容真实性）
- Aqara 编辑团队挑高质量内容做首页推荐（手工运营"明星案例"）

---

## 关键 KPI

| 指标 | 目标 |
|---|---|
| ACB 发布一个 Showcase 的耗时 | **≤ 30 min**（合格线） |
| ACB 月度发布率 | ≥ 60% 活跃 ACB 月发布 ≥ 1 |
| Showcase → Lead 转化率 | 持续提升 |
| Lead → 成交率 | 透明归因 |
| 内容驱动的插件 + Studio 订阅销售 | 占 Marketplace 总销售 ≥ 30% |

---

## 内容质量与速度的平衡

| 层 | 机制 |
|---|---|
| 第一道 | **自动审核**（基础合规 / 隐私 / 敏感） |
| 第二道 | **人工抽检**（质量 / 真实性，与 Studio 部署记录交叉验证） |
| 第三道 | **用户举报 + 评论评分** |

---

## 落地原则

1. Showcase Composer 的核心 KPI 是 **"ACB 发布一个 Showcase 的耗时"**——压到 30 min 以内才算合格。
2. 任何内容相关功能都要内嵌 **"这条内容给你带来了什么"** 的归因看板（Lead / 插件销售 / Studio 部署）。
3. 排名算法要**透明**（ACB 知道发什么、怎么发能涨权重）。
4. 内容质量审核 vs 发布速度的平衡：自动审核 + 人工抽检 + Studio 部署记录交叉验证。
5. 早期手工运营"明星案例"——Aqara 编辑团队挑高质量内容做首页推荐。
6. **Showcase 与 Studio 强绑定**：未部署 Studio 的"案例"不进入官方 Discover 推荐池——保证内容代表真实交付能力。
